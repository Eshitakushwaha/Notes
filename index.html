    Selector in CSS

Selector in CSS:

In CSS, there are 4 types of selector

  • Universal selector

  • Tag selector

  • Id selector

  • Class selector

  • Universal selector:

Universal selector is written by using (*). It will select all the elements.

Ex =
*h1{ color: red; }

  • Tag selector:

In tag selector the name of any tag is written.

Ex =
p{ color: red; }

  • Id selector:

Id selector is written by using (#) with the name of id.

Ex =
#h1{ color: red; }

  • Class selector:

Class selector is written by using (.) with the name of class.

Ex =
.h1{ background-color: red; }

Colors

Colors in CSS:

We can write the color in CSS in the following ways.

  • Color Name [black, blue, white, red, green]

  • Color Code [RGB]

    • There are only three naturals colors [Red, Green, Blue]

    In CSS we can write the rgb value of color between (0-255).

    Ex =
    h1{ color: rgb(0, 255, 0);}

  • Hex Code

    • We can write the hexa decimal of any rgb code.

    Ex =
    h1{ color: rgb(255, 100, 150);
    color: #ff6496";}

  • HSL [Hue, Saturation, Light]

    • HSL is using for editing, photoediting etc.

    Ex =
    h1{ color: hsl(66, 100%, 50%); }
